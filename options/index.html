Параметры AirTab

Управление поиском, темой и резервной копией в отдельной вкладке.

Выберите язык UI. Авто использует язык браузера.

Поисковые системы

Тема и производительность

Кнопка выбора файла привязывает файл по ссылке без загрузки картинки в localStorage.
Важно: если у пространства задан свой фон, он перекрывает глобальный фон темы.
Эко-режим снижает визуальные эффекты и нагрузку на слабых устройствах.
Отключайте в обычной работе. Включайте только для диагностики drag-and-drop.

Резервная копия

Экспортируются все данные AirTab: пространства, папки, ярлыки, панели, поисковики, фоны, режим производительности и DND debug.

Синхронизация

Выберите режим синхронизации:

Dropbox

Автосинхронизация включена: изменения отправляются сразу, получение — при открытии/обновлении вкладки AirTab.
Dropbox не подключен.

Файл (iCloud/Dropbox/папка)

Привяжите один `AirTab.sync.json` в синхронизируемой папке. Это самый легковесный вариант.
Автосинхронизация включена: изменения отправляются сразу, получение — при открытии/обновлении вкладки AirTab.
Файл не привязан.

Импорт закладок

Файл не выбран (.html, .htm, .json)
Выберите файл для предпросмотра импорта.
Поддерживаются экспортированные файлы из Firefox/Chromium/Raindrop (HTML) и JSON-экспорты.
